# Contributing to SkillManagerPro

I'm building this in public and I love collaboration! If you'd like to contribute, here is how:

1. **Issues:** If you find a bug or have a feature idea, please open an issue first. 
2. **Pull Requests:** - Fork the repo.
    - Create a branch for your feature/fix.
    - Submit a PR with a clear description of the changes.

Thanks for helping make this plugin better!
